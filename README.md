## Installation

```bash
$ npm install
```

## Running the app

https://futbol-nest.onrender.com/teams

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
